# AI Business Intelligence Assistant

An AI-powered tool that allows users to analyze structured datasets using natural language queries.

## 🚀 Features
- Upload CSV datasets
- Preview and explore data
- Ask questions about your data (coming soon)
- AI-generated insights (RAG pipeline in progress)

## 🧠 Tech Stack
- Python, Pandas
- Streamlit
- OpenAI / LLMs
- FAISS (Vector DB)
- LangChain
